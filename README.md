## ✨ What is Goody.ai?
Goody.ai is an AI-powered knowledge app that answers any question beautifully — with voice reading, YouTube video references, web sources, and instant quizzes. Designed with a Pinterest-inspired aesthetic for a delightful learning experience.

## 🚀 Features
- 🤖 AI answers any question
- 🔊 Voice reading with 5 voice styles
- 📺 YouTube video references for every answer
- 🔗 Real webpage sources
- 🧠 Auto-generated quiz after every answer
- 🎙️ Change voice — Soft, Calm, Fun, Deep, Fast
- 🌍 Multi-language support
- 📋 Copy and Share answers
- 📖 Simple / Detailed / Fun answer modes
- 🌸 Pinterest aesthetic UI

## 🛠️ Built With
- HTML, CSS, JavaScript
- React.js
- Web Speech API (voice)
- Deployed on Vercel
- Version controlled on GitHub
